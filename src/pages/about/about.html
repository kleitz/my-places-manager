Acerca de My Places Manager
tu gestor de lugares favorito

My Places Manager es una aplicación que permite al usuario buscar información sobre varios tipos de lugares, filtrarlos por diversos parámetros y visualizarlos sobre un mapa.

También le permite gestionar sus propias listas de lugares, desde favoritos, visitados... hasta listas personalizadas.

Para ello, hace uso de las APIs de places y maps de Google.

Desarrollada usando Ionic Framework
